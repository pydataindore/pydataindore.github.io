About PyData

An Educational Program by NumFOCUS

PyData provides a forum for the international community of users and developers of data analysis tools to share ideas and learn from each other. We promote discussion of best practices, new approaches, and emerging technologies.

▸ Open source first — always
▸ Community-driven learning
▸ Inclusive for all skill levels
▸ Giving back to the ecosystem
